Meu Primeiro Projeto HTML

A ideia inicial é desenvolver novas habilidades durante a construção de um pequeno site do zero. Nesse momento, estou estudando por conta dos próprios fundamentos como:

- Estrutura do Documento: (Conceitos de Doctype, HTML, Head e Body);
- Hierarquia e Formatação de Texto: (Uso das Tags h1 a h6, p, br e pre).
Estou utilizando o Visual Studio Code como ferramenta de aprendizado, juntamente com a tecnologia HTML5.

Pretendo atualizar este projeto com novos aprendizados, com foco principal em:

- Tags Estruturais;
- Links e Imagens;
- Listas e Tabelas.
